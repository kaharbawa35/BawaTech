Interpreting the learning curve¶

This learning curve doesn't look quite good. We see that the loss on the training set is relativaly not decreasing for regardless of the epochs. This shows the network is not quite learning to classify the pictures. For the test set the loss does not decrease as fast but stays roughly within the same range above the training curve. This means our model might be underfitting.

Interpreting the learning curve¶

This learning curve doesn't look quite good. We see that the accuracy and loss on both the training and test set is constant and at its maximum irrespective/regardless of the epoch. This shows severe underfitting. There is a huge gap between our training loss and test loss graph. Our model does not have sufficient capacity to handle the complexity of the data. It should be advisable to reduce the complexity of our model through some dropouts or Dense_sparse_dense training.
